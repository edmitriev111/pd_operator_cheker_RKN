Инструкция по использованию программы проверки операторов персональных данных
=========================================================================

1. Подготовка к работе
----------------------
1.1. Установите Python версии 3.7 или выше с сайта https://www.python.org/downloads/
     При установке обязательно поставьте галочку "Add Python to PATH"

1.2. Установите Google Chrome браузер, если он еще не установлен
     https://www.google.com/chrome/

1.3. Установите необходимые библиотеки. Для этого:
     - Откройте командную строку Windows (нажмите Win + R, введите cmd и нажмите Enter)
     - Перейдите в папку с программой (используйте команду cd, например: cd C:\Users\Username\Desktop\pd_checker)
     - Выполните команду: pip install -r requirements.txt
     - Установите дополнительные библиотеки: 
       pip install tk
       pip install pyarrow

2. Подготовка списка ИНН
------------------------
2.1. Создайте файл inn.txt в той же папке, где находится программа
2.2. Внесите в него список ИНН компаний для проверки
2.3. Каждый ИНН должен быть на новой строке
2.4. Пример содержимого файла inn.txt:
     7707083893
     7728168971
     7706107510

3. Запуск программы
------------------
3.1. Откройте командную строку Windows
3.2. Перейдите в папку с программой
3.3. Выполните команду: python pd_checker.py

4. Получение результатов
-----------------------
4.1. После завершения работы программы будут созданы два файла:
     - report.xml - отчет в формате XML
     - report.xlsx - отчет в формате Excel с фильтрами

4.2. Excel-отчет содержит следующие столбцы:
     - ИНН
     - Статус (Найден/Не найден в реестре)
     - Регистрационный номер
     - Наименование
     - Тип оператора
     - Основание включения
     - Дата регистрации
     - Дата начала обработки
     - Ссылка на карточку

4.3. В Excel-отчете доступны:
     - Фильтрация по любому столбцу
     - Сортировка данных
     - Поиск по значениям
     - Автоматическая настройка ширины столбцов

5. Возможные проблемы и их решение
---------------------------------
5.1. Если появляется ошибка "python не является внутренней командой":
     - Переустановите Python с галочкой "Add Python to PATH"
     - Перезапустите командную строку

5.2. Если появляется ошибка при установке библиотек:
     - Проверьте подключение к интернету
     - Попробуйте запустить командную строку от имени администратора

5.3. Если программа не может найти файл inn.txt:
     - Проверьте, что файл находится в той же папке, что и pd_checker.py
     - Проверьте правильность написания имени файла

5.4. Если возникает ошибка с Chrome:
     - Убедитесь, что Google Chrome установлен
     - Попробуйте обновить Chrome до последней версии

5.5. Если не открывается Excel-отчет:
     - Убедитесь, что установлен Microsoft Excel
     - Проверьте, что файл не открыт в другой программе

6. Примечания
------------
- Программа использует автоматизированный браузер Chrome в фоновом режиме
- Между проверками каждого ИНН установлена пауза в 3 секунды
- Программа работает только при наличии подключения к интернету
- Для корректной работы с кириллицей используется кодировка UTF-8
- Excel-отчет содержит автоматические фильтры для удобства анализа данных
